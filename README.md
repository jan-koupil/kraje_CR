# Kraje ČR

příprava pro 3D tisk skládačky „kraje ČR“. Corel zdroják, SVG soubory, dávkový generátor STL (pomocí OpenSCAD). Vytvořené soubory se tváří, že mají inverse facets, ale tisknout se dají bez problémů (případně PrusaSlicer udělá nějaký Windows fix a vše funguje)
Pro menší skládačku je potřeba si udělat asi jiná SVG, bylo by to moc natěsno.
